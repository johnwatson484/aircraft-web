# Aircraft web
View tracked aircraft

## Prerequisites

Either:
- Docker
- Docker Compose

Or:
- Node 16+

## Running application
### Docker
```
docker-compose build
docker-compose up
```

### Node
```
node app
```
Or:
```
nodemon app
```
